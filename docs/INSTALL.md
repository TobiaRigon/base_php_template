# Installazione

## Passaggi

1. Clona il repository:
   ```bash
   git clone https://github.com/tTobiaRigon/base_php_template.git
   cd tuo-progetto
   ```
2. Installa le dipendenze con Composer:
   ```bash
   composer install
   ```
3. Configura il file .env:
   ```bash
   DB_HOST=localhost
   DB_NAME=database
   DB_USER=user
   DB_PASSWORD=password
   ```
